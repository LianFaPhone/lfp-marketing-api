新订单是      刚刚创建的订单

待处理订单    比如国政通不通过的，可能是填错了，要安排人员主动联系，联系完 就是 已处理
已处理订单

已激活订单   号卡归属地 ，（激活导入）已激活订单是否 会和之前的新订单重复
回收站      删除的订单
激活导入     （不要了这个功能）
